# Bunker Privacy Policy

**Last updated: March 2026**

## Data Collection
Bunker stores all decision data locally on your device. Your decisions, criteria, and outcomes are never uploaded to external servers unless you explicitly use cloud sync or share features.

## Local Storage
- All decision data is stored locally using SQLite
- User preferences are stored in UserDefaults
- No personal data is transmitted without explicit consent

## AI Features
- AI decision advice is processed through Apple's on-device AI frameworks where available
- No decision data is sent to external AI services without user action

## Calendar Integration
- Bunker requests calendar access to create decision deadline events
- Events are created in a dedicated "Bunker Decisions" calendar
- Calendar data is never uploaded or shared

## Analytics
- Anonymous usage analytics help us improve the app
- No personally identifiable information is collected
- Users can opt out at any time

## Third-Party Services
- None required for core functionality

## Contact
For privacy concerns, contact: privacy@bunker.app
